# CatlGLM

`CatlGLM` packages contains below functions:

- `CatlGLM_linear()`
- `CatlGLM_binomial()`
- `CatlGLM_inf()`

The package implements adaptive transfer learning for compositional covariates under a zero-sum constraint for linear and logistic regression, plus de-biased confidence intervals.

## Installation

After unzipping the package source, install from the package directory with:

```r
install.packages("path/to/CatlGLM_pkg", repos = NULL, type = "source")
```
